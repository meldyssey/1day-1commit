# 1day-1commit

프론트엔드 개발 스터디 & 학습 기록 저장소

## 목표

- 하루 최소 1회 커밋으로 꾸준한 학습 습관 형성
- 알고리즘 문제 해결 능력 향상
- JavaScript/React 개념 이해 및 실습

## 주요 학습 내용

### 📚 알고리즘 문제 풀이

- **baekjoon/**: 백준 온라인 저지 문제 풀이
- **coding-test/**: 코딩 테스트 기출 문제 및 연습

### ⚛️ React 기능 구현

- **react-test/**: React 컴포넌트 및 기능 구현 실습
  - Counter, TodoList 등 기본 기능 구현
  - React Hooks 활용 연습

### 💻 JavaScript 개념 학습

- 비동기 프로그래밍 (async/await)
- 암호화 (crypto-js 활용)
- 알고리즘 & 자료구조

## 커밋 형식

- 알고리즘 문제풀이: `coding-test: 작업 내용`
- 자바스크립트 학습: `js-study: 작업 내용`
- 리액트 기능 구현: `react-test: 작업 내용`
- Next.js 기능 구현: `next-test: 작업 내용`
